## 🚀 运行测试

### 基本命令

```bash
# 激活虚拟环境
source venv_ai/Scripts/activate

# 运行所有测试
pytest

# 运行单元测试
pytest tests/unit/

# 运行集成测试
pytest tests/integration/

# 查看详细输出
pytest -v -s

# 生成覆盖率报告
pytest --cov=. --cov-report=html
```

### Windows 用户

```bash
# 直接运行测试脚本
tests/run_tests.bat
```

- 测试数据库在测试完成后自动清理

  - 虚拟环境是否正确激活
