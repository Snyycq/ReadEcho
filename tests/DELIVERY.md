## 🚀 如何使用

### 快速开始

```bash
# 激活虚拟环境
source venv_ai/Scripts/activate

# 运行所有测试
pytest

# 查看详细输出
pytest -v -s

# 生成覆盖率报告
pytest --cov=. --cov-report=html
```

### 运行特定测试

```bash
# 运行单元测试
pytest tests/unit/

# 运行集成测试
pytest tests/integration/

# 运行特定模块测试
pytest tests/unit/test_validators.py

# 运行上次失败的测试
pytest --last-failed
```
